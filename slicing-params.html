工艺（Process）

本章为 Creality Print 切片工艺参数的顶层模块，涵盖从模型切片到 G-code 输出的完整工艺链路。以下各章节按功能域划分，包含质量、强度、速度、支撑、材料及其他辅助参数，以及参数联动关系和典型场景推荐。

第一章 质量（Quality）

加热仓功能需要硬件支持（密封仓体 + 加热元件 + 温控传感器），并非所有 Ender/K 系列机型均支持。

7.3 打印温度（Print Temperature）

本节控制喷嘴（热端）温度，是 FDM 打印最核心的耗材参数，直接影响层间结合、表面质量、挤出稳定性和材料力学性能。

序号参数名称英文名称定义典型取值实际影响场景
7.3.1打印温度Printing Temperature正常打印层的喷嘴温度180–300℃（按材料类型）最核心耗材参数——PLA 190-210℃；PETG 230-250℃；ABS 250-270℃；温度影响粘度、流动性、层间结合和拉丝程度
7.3.2打印温度（首层）Printing Temperature Initial Layer第一层打印的专用温度正常温度 -5℃ 至 +10℃首层略高可提升附着力，但过高会导致首层变形；建议比正常温度高 5℃
7.3.3打印温度（底面）Printing Temperature Bottom底部几层的专用温度正常温度 -5℃ 至 +5℃可独立于主体设置，控制底部几层的挤出特性
7.3.4打印温度（逐层）Printing Temperature Layer允许每层单独设置不同的打印温度开启 / 关闭开启后可添加多段温度变化的 G-code；用于高级工艺控制（如预热层、回温层）
7.3.5初始温度等待Initial Temperature Waiting打印前等待喷嘴达到目标温度开启 / 关闭开启确保喷嘴完全预热后再开始挤出，防止冷启动堵头
7.3.6腔室预热温度Chamber Temperature在打印开始前预热仓室的目标温度0–60℃对于 ASA/ABS，提前预热打印仓可减少收缩和翘边风险
7.3.7保温时间Dwell Time加热完成后开始打印前的停留时间0–30 分钟0=立即开始；延长保温时间可使热床和仓室温度更均匀，减少热胀冷缩导致的应力

打印温度是最影响打印质量与材料性能的耗材参数。建议首次使用时从材料供应商推荐范围的中值开始，根据实际效果（拉丝、层间分离、过热气泡）上下微调 5-10℃。

7.4 床温（Bed Temperature）

热床温度控制打印件与平台的附着，影响首层粘附、翘边风险和底部表面质量。不同材料对热床温度的需求差异显著。

序号参数名称英文名称定义典型取值实际影响场景
7.4.1热床温度Bed Temperature正常打印阶段的热床目标温度30–110℃（按材料类型）决定首层附着力的核心参数——PLA 50-60℃；PETG 70-80℃；ABS/ASA 90-110℃；温度过低=首层不粘；温度过高=底面拉丝/变形
7.4.2热床温度（首层）Bed Temperature Initial Layer第一层打印的热床温度正常温度 ±0–20℃首层略高可增强附着力；ABS 建议比正常高 5-10℃
7.4.3热床温度（底面）Bed Temperature Bottom底部几层的热床温度正常温度 ±0–10℃可独立于主体设置，控制底部几层的热粘附特性
7.4.4热床温度（逐层）Bed Temperature Layer允许每层单独设置不同的热床温度开启 / 关闭开启后可添加热床温度渐变 G-code；用于高级工艺控制
7.4.5初始热床温度等待Initial Bed Temperature Waiting打印前等待热床达到目标温度开启 / 关闭开启确保热床完全预热后再开始打印，防止首层因温度不均附着不良

热床温度与喷嘴温度共同决定了 FDM 打印的"热环境"。对于 ABS/ASA 等高收缩材料，建议热床温度设为材料玻璃化转变温度（7.1.4.3）附近，以减少层间应力。

7.5 体积速度限制（Volumetric Speed Limit）

体积速度（Volumetric Speed）= 线宽 × 层高 × 打印速度，决定了单位时间内挤出的耗材体积。此参数限制了打印机在给定耗材类型下的最大安全挤出量，防止挤出不足或过挤导致的打印缺陷。

序号参数名称英文名称定义典型取值实际影响场景
7.5.1最大体积速度Maximum Volumetric Speed允许的最大耗材挤出体积速度（mm3/s）3–30 mm3/s（按材料和挤出机类型）保护挤出系统的关键参数——直驱挤出机可达 15-20mm3/s；Bowden 通常 8-12mm3/s；超过此值会导致挤出不足、层纹、缺料
7.5.2温度首层过高时降低速度Slow Down if Temperature Too Low当喷嘴温度低于目标时自动降低打印速度开启 / 关闭开启可防止温度波动导致的挤出不稳定，在加热不足时减少缺陷
7.5.3温度过低时的速度系数Slow Down Factor if Temperature Too Low温度不足时速度降低的比例50%–100%值越低=温度不足时降速越多，防止挤出不足

最大体积速度是高速打印的核心限制参数。K2 Pro 等高速机型需要配合高流量热端设计才能发挥优势。用户在调高打印速度时，应同步检查此参数是否成为瓶颈。

第八章 冷却（Cooling）

8.1 特定层冷却（Specific Cooling）

本节控制特定层（如首层、特定高度层）的冷却风扇策略，用于优化不同打印阶段的散热需求。

序号参数名称英文名称定义典型取值实际影响场景
8.1.1启用特定层风扇Enable Specific Layer Cooling允许对特定层单独设置风扇策略开启 / 关闭开启后可添加多层不同的风扇策略，用于精细控制冷却
8.1.2特定层冷却表Specific Layer Cooling Table定义特定层对应的风扇转速列表层数 + 风扇转速 (%)可设置首层 0%、第 5 层 30%、第 10 层 100% 等逐层策略
特定层冷却通常用于多层材料打印或需要在打印中途切换风扇策略的高级场景。

8.2 模型风扇（Model Cooling Fan）

模型风扇（part cooling fan）是控制挤出后耗材冷却速度的核心设备。适当的冷却可防止 PLA 等材料过热变形、保持细节精度，并影响层间粘附。

序号参数名称英文名称定义典型取值实际影响场景
8.2.1模型风扇Model Cooling Fan模型冷却风扇的启用开关开启 / 关闭影响整体散热策略——关闭后全程无模型冷却，仅使用机箱风扇散热；用于 ABS 等需要慢速冷却的材料
8.2.2最大风扇速度Maximum Fan Speed模型风扇的最高转速限制50%–100%100%=全速；降低可防止 PLA 首层因冷风冲击而翘边；高细节打印建议限制在 80%
8.2.3最小风扇速度Minimum Fan Speed模型风扇的最低转速限制0%–50%非零值保证基础散热；PLA 建议 ≧30%；高流量打印建议提高最低速度以防过热
8.2.4节流风扇速度Throttled Fan Speed当 CPU/主板温度过高时自动降低风扇转速0%–100%保护电子元件，防止因过热导致固件降频或重启；建议设置 50-70%
8.2.5风扇总在最底层启动Fan Always On from Layer Zero从首层开始即启用模型风扇开启 / 关闭关闭=首层风扇关闭以确保附着；开启=首层即开始散热，适用于特定的高速打印场景
8.2.6首层风扇速度Initial Layer Fan Speed第一层打印时的风扇速度0%–100%通常设为 0% 或低转速以保证首层附着；高速打印时可能需要略高转速
8.2.7首层冷却激活高度First Layer Cooling Activation Height首层后开始增加风扇的高度阈值0.1–5 mm确保首层固化后再开始吹风；通常设为 0.5-1mm
8.2.8冷却层开始速度Layer Start Fan Speed每层开始打印时的初始风扇转速0%–100%控制每层重启时的风扇响应行为
8.2.9冷却速度Cooling Fan Speed正常打印层的风扇转速（层高 > 冷却激活高度）0%–100%最核心的冷却参数——PLA 推荐 50-100%；PETG 推荐 20-50%（PETG 怕过度冷却）；ABS 推荐 0-20%
8.2.10冷却速度逐层变化Cooling Fan Speed by Layer允许风扇速度随层高变化开启 / 关闭开启后可设置风扇速度随高度渐进调整（如底部低、顶部高）
8.2.11冷却速度表Cooling Fan Speed Table定义不同层高对应的风扇转速表层高 + 风扇转速精细控制不同高度的冷却策略
8.2.12桥接风扇速度Bridging Fan Speed悬空桥接时的风扇转速100%（满速）桥接时满速冷却可减少下垂，提升桥接质量

冷却参数是材料特性差异最大的参数组。PLA 需要强冷却（50-100%），ABS 需要几乎无冷却（0-20%），PETG 介于两者之间（20-50%）。Creality Print 应根据耗材类型自动推荐默认值。

8.3 辅助风扇（Auxiliary Cooling Fan）

辅助风扇用于补充模型风扇的冷却能力，或为打印件的特定区域提供定向冷却。

序号参数名称英文名称定义典型取值实际影响场景
8.3.1启用辅助风扇Enable Auxiliary Fan启用辅助风扇设备开启 / 关闭开启后可配置第二个冷却风扇；用于大型打印机或需要定向冷却的场景
8.3.2辅助风扇速度Auxiliary Fan Speed辅助风扇的目标转速0%–100%独立于模型风扇控制；可用于对准特定区域吹风
8.3.3辅助风扇同步模型风扇Sync Auxiliary with Model Fan辅助风扇与模型风扇同步转速开启 / 关闭开启后辅助风扇速度跟随模型风扇变化；关闭后独立控制
辅助风扇需要硬件支持（部分 Creality 机型提供 AUX 接口）。

8.4 机箱风扇（Chamber Cooling Fan）

机箱风扇控制打印机腔体（机箱）内部的空气循环，用于维持稳定的打印环境温度，与模型冷却风扇有本质区别。

序号参数名称英文名称定义典型取值实际影响场景
8.4.1机箱风扇Chamber Cooling Fan机箱风扇的启用开关开启 / 关闭用于维持仓温稳定——开启后持续循环机箱内空气；ABS/ASA 打印时可能需要关闭以保持高温环境
8.4.2最小腔室温度Minimum Chamber Temperature触发机箱风扇开启的最低仓温阈值0–80℃仅当仓温高于此值时开启机箱风扇，用于高温材料打印时防止过热
8.4.3腔室温度过高时降低打印速度Slow Down if Chamber Temperature Too High当仓温超过阈值时降低打印速度开启 / 关闭开启可防止高温导致的打印质量问题
8.4.4温度过高时的速度系数Speed Factor if Chamber Temperature Too High仓温过高时速度降低的比例50%–100%值越低=仓温过高时降速越多

机箱风扇与模型风扇的功能定位不同：模型风扇用于快速冷却挤出丝材（影响层间结合和细节）；机箱风扇用于控制整体环境温度（影响翘边和收缩）。两者需要配合使用。

第九章 参数覆盖（Overrides）

9.1 回抽覆盖（Retraction Overrides）

本节允许对特定耗材单独覆盖工艺模块中的回抽设置。当同一打印机使用多种耗材时，不同材料可能需要不同的回抽策略（如 TPU 需要更少的回抽）。

序号参数名称英文名称定义典型取值实际影响场景
9.1.1启用回抽覆盖Enable Retraction Overrides启用耗材级别的回抽参数覆盖开启 / 关闭开启后允许此耗材使用独立的回抽设置，而非继承工艺模块的全局设置
9.1.2回抽距离Retraction Distance回抽时耗材被拉回的长度（mm）0–10 mm（典型 2-6mm）控制拉丝的核心参数——PLA 3-5mm；PETG 4-6mm；TPU 0-1mm（软材几乎不回抽）；Bowden 挤出机需要更长距离
9.1.3回抽速度Retraction Speed回抽动作的执行速度（mm/s）15–60 mm/s速度越快回抽越干净但可能磨损耗材；软材建议慢速（15-25mm/s）；硬材可快速（30-60mm/s）
9.1.4额外回抽距离Extra Retraction Distance每次连续回抽之间的额外回抽长度0–5 mm用于减少连续短距离移动后的拉丝
9.1.5最小回抽挤出头温度Minimum Extrusion Temperature for Retraction允许回抽的最低喷嘴温度180–250℃低温时禁用回抽可防止因材料粘度大导致的回抽失败和堵头；PLA 通常设为 190℃
9.1.6回抽最小行程Retraction Minimum Travel移动距离超过此值时才触发回抽0–10 mm设置此值可避免短距离空驶也触发回抽，减少回抽次数，延长挤出机寿命
9.1.7最大回抽计数Maximum Retraction Count每毫米行程允许的最大回抽次数0–200 次/mm限制过高频率的回抽，保护挤出机构；0=不限制
9.1.8回抽频率限制启用Retraction Frequency Limit启用回抽频率限制开启 / 关闭开启后限制每毫米行程的回抽次数，防止过度回抽
9.1.9高速回抽距离High Speed Retraction Distance高速打印时的回抽距离0–10 mm高速打印时可能需要更长的回抽距离以补偿惯性
9.1.10高速回抽速度High Speed Retraction Speed高速打印时的回抽速度30–100 mm/s高速挤出时回抽也需要更快速度以匹配流量
9.1.11高速回抽启用阈值High Speed Retraction Enable Threshold触发高速回抽策略的打印速度阈值50–200 mm/s当打印速度超过此值时启用高速回抽参数
9.1.12回抽提笔Retraction Lift回抽时 Z 轴抬起的额外高度0–5 mm回抽时轻微抬喷嘴可减少碰撞未固化材料的风险；极短抬升（0.1-0.2mm）可有效防止拉丝
9.1.13启用 Z hop 回抽Enable Retraction Z Hop回抽时启用 Z hop（Z 轴抬起）开启 / 关闭开启后回抽时喷嘴轻微抬起，避免划过未固化层；有助于打印高细节模型
9.1.14回抽 Z hop 高度Retraction Z Hop Height回抽时 Z 轴抬起的高度0.1–2.0 mm一般 0.2-0.5mm 足够；过高会明显增加打印时间
9.1.15回抽后延迟Retraction Retract Before Wipe擦拭动作前的回抽延迟0–200 ms控制回抽与擦拭动作之间的时序
9.1.16多材回抽Multi Material Retraction多材料打印时的回抽策略开启 / 关闭开启后多材料切换时使用专用的多材料回抽距离
9.1.17多材料回抽距离Multi Material Retraction Distance多材料切换时的专用回抽距离0–10 mm通常比普通回抽更长，因为多材料系统的喉管更长
9.1.18启用"部分回抽"Enable Partial Wipe启用部分回抽（擦拭而非完全回抽）开启 / 关闭开启后可设置部分回抽代替完全回抽，减少回抽次数和时间
9.1.19部分回抽距离Partial Wipe Distance部分回抽（擦拭）的距离0–10 mm小于完全回抽距离，仅将耗材拉回喷嘴附近而非完全回抽

回抽参数是 FDM 打印中最影响拉丝和渗料的耗材级参数。TPU、PVA 等软性/可溶性材料需要显著减少甚至禁用回抽。回抽策略应作为耗材预设的一部分，而非每次单独调整。

第十章 高级（Advanced）

10.1 耗材丝 G-code（Filament G-code）

本节允许为特定耗材添加自定义 G-code 指令，在打印开始前、结束后或特定层执行，用于材料特定的初始化、后处理或特殊功能触发。

序号参数名称英文名称定义典型取值实际影响场景
10.1.1启用耗材丝 G-codeEnable Filament G-code启用此耗材的自定义 G-code开启 / 关闭开启后可为此耗材添加专属 G-code 脚本；常用于非标准材料或特殊打印机配置

耗材 G-code 是高级定制功能，用于支持特殊材料（如尼龙需预干燥、PC 需高功率灯丝）和自动化工作流。应提供常用材料的预设模板。

11.1.5冷却变色区域Cool Down Color Change Area换色前喷嘴降温的区域开启 / 关闭开启后换色前将喷嘴降温，减少渗料；但会增加换色时间
11.1.6冷却变色温度Cool Down Temperature for Color Change换色前的目标降温温度150–220℃PLA 降至约 180℃；PETG 降至约 200℃；过高会渗料，过低可能堵头
11.1.7变色后恢复温度Temperature Restore After Color Change换色后恢复的目标温度180–260℃换色后迅速恢复至正常打印温度，保证挤出流畅
11.1.8自动恢复温度Auto Temperature Restore自动恢复打印温度开启 / 关闭开启后自动在变色后升温；关闭后需手动控制
11.1.9逐层换色Color Change per Layer每层可单独设置颜色切换开启 / 关闭开启后可定义每层的换色点；用于复杂多色模型
11.1.10换色层数Color Change Layers定义每层的换色点列表层号 + 耗材索引可设置任意层切换到任意颜色的耗材
11.1.11换色 Z 高度Color Change Z Height在特定 Z 高度触发换色自定义高度列表基于绝对 Z 高度而非层数设置换色点
11.1.12最后一层不换色No Color Change at Top顶层区域禁用换色开启 / 关闭开启后可防止顶层出现换色导致的表面瑕疵
11.1.13最后一层不换色的厚度Top Distance for Color Change顶层区域不触发换色的厚度0–5 mm确保顶层外观完整，避免换色痕迹影响展示面
11.1.14最后一层不换色的优先级Top Color Change Priority顶层换色限制的优先级0–10值越高=顶层换色限制越严格

多材料换色是多色打印体验的核心，直接影响换色痕迹的可见性和颜色纯净度。快速换色（11.1.3）仅适用于相近颜色切换；反差大的颜色（如黑→白）应使用标准换色策略并增加清理量。
